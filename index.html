Code Mania 110

Moving Forwards, Open Source World

วันเสาร์ที่ 25 พฤศจิกายน 2560

คณะเทคโนโลยีสารสนเทศ มหาวิทยาลัยพระจอมเกล้าธนบุรี (บางมด)

ในหลากหลายสิ่งนั้นในปัจจุบันล้วนถูกขับเคลื่อนด้วย Open Source Project ฉะนั้น Code Mania ในครั้งนี้จึงอัดแน่นไปด้วย Session ที่ล้วนเกี่ยวข้องกับ Open Source ในด้านต่างๆ ตั้งแต่ Philosophy, แนะนำ Open Source Project ที่จะช่วยให้การทำงานในแขนงต่างๆ ไม่ว่าจะเป็น Frontend, Backend, Infrastrcutre และ Security ง่ายขึ้น ตลอดจนแนวทางในการร่วมเป็นส่วนหนึ่งในการ Contribute Open source project โดยในครั้งนี้ได้รับเกียรติจากเหล่าวิทยากรผู้เชี่ยวชาญจากหลากหลายแขนง ที่จะมาถ่ายทอดวิชาแบบลงลึกถึง technical/code กันภายในงาน
